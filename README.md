# Screen Solver Bot 🖥️✨

**Screen Solver Bot** — це інтелектуальний асистент для автоматичного розв'язання тестів та технічних завдань безпосередньо з екрана вашого комп'ютера. Проект розроблений з акцентом на повну конфіденційність, безшумність та швидкість роботи.

---

## 🌟 Основні можливості

1. **Миттєве розв'язання за гарячою клавішею (`Ctrl + Shift + F12`):**
   Достатньо натиснути комбінацію клавіш, щоб утиліта автоматично зробила знімок екрана, надіслала його до ШІ та відобразила правильні відповіді.
2. **Невидимий оверлей на екрані (Capture Exclusion):**
   Відповіді відображаються у красивому напівпрозорому віконці прямо поверх усіх програм. Завдяки використанню Windows API (`SetWindowDisplayAffinity`), це вікно та його контекстне меню **повністю невидимі для будь-якого софту захоплення екрана** (записи OBS Studio, трансляції в Discord / Zoom / MS Teams, скріншоти та системи прокторингу).
3. **Повний Stealth-режим:**
   Програма працює у фоні, не створює консольних вікон, іконки в системному треї чи на панелі завдань.
4. **Підтримка тестів із кількома правильними варіантами:**
   ШІ автоматично розпізнає тип запитання (наприклад, чекбокси) та виводить кожну правильну відповідь суворо з нового рядка без зайвих пояснень.
5. **Дублювання в Telegram:**
   Окрім оверлею, правильні відповіді та копія скріншоту надсилаються до вашого приватного Telegram-бота. Також підтримуються контекстні діалоги — ви можете відповісти на повідомлення бота в Telegram, щоб поставити додаткове запитання щодо скріншоту.

## 🛠️ Інструкція з налаштування

### 1. Підготовка файлу конфігурації
Створіть файл `.env` у корені проекту (або скопіюйте його з прикладу `.env.example`) та впишіть туди свої ключі:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
GEMINI_API_KEY=ваш_api_ключ_openrouter_або_gemini
GEMINI_MODEL=google/gemini-3.5-flash
REASONING_EFFORT=none
HOTKEY=<ctrl>+<shift>+<f12>
```

> [!TIP]
> **Використання інших моделей (GPT-4o, Claude 3.5, Llama тощо):**
> Оскільки наш бот підтримує роботу через **OpenRouter**, ви не обмежені лише моделями Gemini! Якщо ви використовуєте API-ключ OpenRouter, ви можете легко перемикатися на будь-яку іншу модель у світі, яка вміє аналізувати зображення. Для цього просто змініть назву моделі в полі `GEMINI_MODEL` у файлі `.env` на потрібний ідентифікатор з сайту OpenRouter, наприклад:
> * `openai/gpt-4o` (флагманська модель від OpenAI)
> * `openai/gpt-4o-mini` (дуже швидка та дешева модель від OpenAI)
> * `anthropic/claude-3.5-sonnet` (топ-модель від Anthropic з чудовою логікою)
> * `meta-llama/llama-3.2-11b-vision-instruct` (чудова бюджетна open-source модель)

### 2. Встановлення залежностей
Переконайтеся, що у вас встановлено Python 3.10+ та виконайте команду в терміналі:
```bash
pip install -r requirements.txt
```

### 3. Де брати ключі для налаштування?

#### 1️⃣ Telegram Bot Token (`TELEGRAM_BOT_TOKEN`)
1. Відкрийте Telegram, знайдіть офіційного бота [@BotFather](https://t.me/BotFather).
2. Надішліть команду `/newbot`.
3. Введіть назву для вашого бота, а потім унікальний юзернейм (який має обов'язково закінчуватися на `_bot`, наприклад `MyHelperSolver_bot`).
4. **BotFather** надішле вам токен (довгий набір літер і цифр). Скопіюйте його в поле `TELEGRAM_BOT_TOKEN` у файлі `.env`.

#### 2️⃣ Telegram Chat ID (`TELEGRAM_CHAT_ID`)
1. Знайдіть вашого створеного бота в Telegram за його юзернеймом та натисніть кнопку **`Start`** (або надішліть будь-яке повідомлення).
2. Знайдіть у Telegram бота [@userinfobot](https://t.me/userinfobot) та натисніть `/start`.
3. Бот у відповідь покаже ваш числовий **Id**. Скопіюйте його в поле `TELEGRAM_CHAT_ID` у файлі `.env`.

#### 3️⃣ API-ключ Gemini / OpenRouter (`GEMINI_API_KEY`)
Ви можете підключити як прямий ключ від Google Gemini, так і через OpenRouter:
* **Варіант А (OpenRouter - рекомендується, якщо є проблеми з оплатою карткою):**
  1. Зайдіть на сайт [openrouter.ai](https://openrouter.ai/).
  2. Зареєструйтеся та перейдіть у розділ **Keys** $\rightarrow$ **Create Key**.
  3. Скопіюйте створений ключ (починається на `sk-or-...`) в поле `GEMINI_API_KEY` у `.env`.
  4. *Примітка:* Не забудьте поповнити баланс OpenRouter (можна через Google Pay або картку від 5$).
* **Варіант Б (Прямий Gemini API):**
  1. Зайдіть на [Google AI Studio](https://aistudio.google.com/).
  2. Натисніть **Get API Key** $\rightarrow$ **Create API Key**.
  3. Скопіюйте ключ (починається на `AIzaSy...`) в поле `GEMINI_API_KEY` у `.env`.

## 🚀 Використання

Для максимальної зручності керування здійснюється за допомогою миші через два файли сценаріїв:
* **Запуск бота:** Двічі клацніть на **`run_stealth.bat`**. Консоль спалахне на секунду і закриється — бот запущений безшумно у фоні, а на екрані з'явиться стартове віконце *"Чекаю запит..."*.
* **Зупинка бота:** Двічі клацніть на **`stop_bot.bat`**, щоб повністю завершити всі фонові процеси програми.

### 🎮 Керування оверлеєм на екрані:
* **Переміщення:** Затисніть ліву кнопку миші в будь-якому місці оверлею та перетягніть його.
* **Швидке очищення:** Двічі клацніть лівою кнопкою миші по оверлею, щоб стерти відповідь і згорнути вікно.
* **Режим «Клік наскрізь» (Click-Through):** Робить вікно «прозорим» для миші (можна клікати крізь нього по кнопках тестів). Увімкнути/вимкнути можна глобальною комбінацією клавіш **`Ctrl + Shift + F10`** або через праве меню. У режимі кліку наскрізь заголовок вікна змінюється на 🔒, а колір стає помаранчевим.
* **Регулювання прозорості (Ctrl + Scroll):** Затисніть клавішу `Ctrl` та прокручуйте коліщатко миші над оверлеєм, щоб зробити його більш тьмяним або яскравим (діапазон прозорості від 15% до 100%). Оверлей покаже тимчасовий HUD-індикатор рівня прозорості.
* **Кастомне меню (правою кнопкою миші):**
  * *Очистити* — скинути текст.
  * *Сховати на 5 сек* — тимчасово приховати вікно.
  * *Клік наскрізь* — перемкнути режим прозорості для миші.
  * *Закрити оверлей* — згорнути оверлей (програма продовжить працювати у фоні).

## 📂 Структура проекту

* `main.py` — ініціалізація, реєстрація гарячих клавіш та обробка скріншотів.
* `overlay.py` — графічне вікно оверлею з кастомним меню та захистом від захоплення екрана.
* `bot.py` — обробка Telegram Bot API (надсилання результатів, обробка відповідей та сесії).
* `solver.py` — клієнт інтеграції з ШІ (Gemini SDK та OpenRouter API).
* `config.py` — завантаження та валідація налаштувань.
* `run_stealth.bat` / `stop_bot.bat` — скрипти швидкого безшумного запуску та зупинки.
* `.gitignore` — захист вашого файлу `.env` від випадкового завантаження на GitHub.
